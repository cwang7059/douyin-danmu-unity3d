# 游戏场景开发过程记录

更新时间：2026-05-28

本文记录“草地、远山、落日、海岸边”场景替换的实际开发过程，便于后续继续排查和迭代。

## 2026-05-28 场景素材替换

### 目标

- 替换现有厚方块地面，改成草地质感的战场地面。
- 去掉远处突兀的矩形方块山脊，改成远山轮廓。
- 增加落日氛围、海面、海岸沙地。
- 保留当前中世纪村庄、建筑阻挡、道路通行带和单位避障逻辑。

### 素材选择

本阶段选择 Poly Haven CC0 素材：

| 用途 | 素材 | 页面 | 项目路径 |
| --- | --- | --- | --- |
| 草地/苔石地面 | Aerial Grass Rock | https://polyhaven.com/a/aerial_grass_rock | `Assets/Resources/Environment/Online/grass_meadow.jpg` |
| 海岸沙地/道路质感 | Coast Sand 01 | https://polyhaven.com/a/coast_sand_01 | `Assets/Resources/Environment/Online/coast_sand.jpg` |
| 落日海岸天空 | Cape Hill HDRI | https://polyhaven.com/a/cape_hill | `Assets/Resources/Environment/Online/cape_hill_sunset.hdr` |

归档路径：

- `Assets/DownloadedAssets/Environment/Online/PolyHavenCoastalSunset`

### 实现计划

1. 新增运行时环境贴图资源常量。
2. 在 `CreateCoreScene()` 中切换为落日天空盒、暖色方向光和更柔和的雾色。
3. 把 `CreateGround()` 从厚 Cube 改为贴图 Plane，减少“地面大方块”观感。
4. 把 `CreateTerrainDepth()` 的远景方块替换为：
   - 海面 Plane
   - 海岸沙地 Plane
   - 半透明浅水/反光 Plane
   - 远山 Mesh 轮廓
   - 落日太阳盘和地平线光晕
5. 把主路/广场从厚 Cube 改为贴地 Plane，继续保留 `AddRoadCorridor()`。
6. Unity 构建验证，清理临时日志。

### 已完成实现

- 已下载并归档 Poly Haven 草地、海岸沙地、落日海岸 HDRI。
- 已新增运行时环境资源路径常量。
- 已将主相机改为 Skybox 清屏，并应用落日天空盒。
- 已调整方向光、环境光和雾色为暖色落日氛围。
- 已将主地面从厚 Cube 改为带草地贴图的 Plane。
- 已将主街、坦克道、市场广场、横街、怪物入口改为贴地 Plane，保留原有 `AddRoadCorridor()`。
- 已将人族/怪物前线色块从薄 Cube 改为半透明贴地 Plane。
- 已把远景方块替换为海岸沙地、海面、浅水、落日反光、远山轮廓、太阳盘和地平线光晕。
- 已执行 Unity Windows Player 构建验证：`Build Finished, Result: Success.`
- 已删除本次构建临时日志。

### 注意事项

- 新增视觉地面和远景不参与建筑阻挡。
- 道路视觉形态可以换，但通行带数据不能丢。
- 远山和海岸属于背景，不要放到单位可达区域。
- 不再添加零散地面方块、碎石方块、随机厚片作为装饰。
